## Install

```sh
yarn add qbadges
```

## Usage

```js
const { qbadges } = require("qbadges");
const profileId = "<Qwiklabs profile id>";

var badges = async () => {
  var qb = await qbadges(profileId);
  console.log(qb);
};

badges();
```
